# How to use

#####Setting

* Create ptv.js file in config folder

```javascript
module.exports = {
  // Put your PTV keys here
  devId: '',
  devSecret: '',
}
```
* Configure database in dev.js, test.js, prod.js files

#####Run express server

```
npm install
npm start
```

#####Run test

```
npm test
```
